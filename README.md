Start the server:
```
npm init -y
npm install express
node server.js
```


## Folder structure
```
project-root/
├── bin/
│   └── server
├── src/
│   ├── index.html
│   ├── notes/
│   │   └── *.md (Markdown files)
│   └── server.js
├── package.json
└── node_modules/
```

Don't forget 
```
chmod +x bin/server
```
